# 🚀 Smart Travel Planning System using Graph Algorithms

## 📌 Overview

This project is a **C-based travel planning system** that helps users find the **shortest path between cities** and calculate the **total cost of a round trip** based on fuel consumption.

It uses **graph data structures** and **Dijkstra’s algorithm** to efficiently compute optimal routes.

---

## 🎯 Features

* 🗺️ Create a custom city map (graph-based)
* 🔗 Add connections between cities with distances
* 📊 Display the complete graph
* 🚀 Find shortest path between two cities
* ⛽ Calculate round-trip fuel cost
* 💰 Cost estimation based on mileage & fuel price

---

## 🧠 Concepts Used

* Graph (Adjacency List Representation)
* Dijkstra’s Algorithm
* Dynamic Memory Allocation
* Structures in C
* String Handling
